# ApiTelegramStore 🛒

Бэкенд-приложение (REST API) для Telegram-магазина цифровых товаров и подписок.

## Стек технологий
* **Java 21**
* **Spring Boot 3.x** (Web, Data JPA)
* **PostgreSQL**
* **Lombok**

## Текущий функционал
* Полный CRUD для управления товарами (`Product`)
* Автоматическая генерация таблиц в БД через Hibernate
* Глобальный перехват ошибок (`GlobalExceptionHandler`)
* Бизнес-логика покупки товара с проверкой остатков на складе
